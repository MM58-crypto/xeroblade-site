# Xeroblade!

Personal site built with Hugo 
---

## Requirements

- [Hugo](https://gohugo.io/) (extended version recommended)
- Git

---

## Installing Hugo

### Ubuntu / Linux Mint

```bash
sudo apt update
sudo apt install hugo
```

Or via Snap:

```bash
sudo snap install hugo
```

### Arch Linux

```bash
sudo pacman -S hugo
```

---

## Getting Started

Clone the repo:

```bash
git clone https://github.com/MM58-crypto/xeroblade-site 
cd xeroblade-site
```
---

## Running Locally

Start the development server:

```bash
hugo server
```

Then open `http://localhost:1313` in your browser. The site hot-reloads on any file change.

To include draft posts:

```bash
hugo server -D
```
